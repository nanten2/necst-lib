neclib.interfaces.console_logger¶

getLogger(name, file_path, obslog_file_path, min_level=10)[source]¶

Get logger instance which prints operation logs to console and dumps to file.

Parameters

  • name (str) – Name of the logger. Calling this function with same name returns the same logger instance.

  • file_path (Union[PathLike, str, bytes]) – Path to file into which all logs (severity >= logging.DEBUG) are dumped.

  • obslog_file_path (Union[PathLike, str, bytes]) – Path to file to log observation summary.

  • min_level (int) – Lower bound of severity level to be displayed on terminal. To suppress less severe messages, set higher value. No matter this value, the log file contains all messages severer than logging.DEBUG (level=10).

Return type

ConsoleLogger

Examples

>>> logger = neclib.interfaces.getLogger("OTF_observation", "path/to/log.txt")
>>> logger.debug("Inform something only needed on diagnosing some problem.")
>>> logger.info("Inform something EXPECTED has happened.")
>>> logger.warning("Inform something user should care has happened.")
>>> logger.error(f"Notify some {functionality} cannot be completed due to error.")
>>> logger.critical(
...     "Notify the %s cannot continue operation due to %s", "program", "some error"
... )
>>> logger.obslog("Observation finished", indent_level=1)
