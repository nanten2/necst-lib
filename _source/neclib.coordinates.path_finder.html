Parameters:

  • location (EarthLocation) – Location of observatory.

  • pointing_param_path (Optional[PathLike]) – Path to pointing parameter file.

  • pressure (Optional[Quantity]) – Atmospheric pressure at the observation environment.

  • temperature (Optional[Quantity]) – Temperature at the observation environment.

  • relative_humidity (Optional[Union[int, float, ndarray, Quantity]]) – Relative humidity at the observation environment.

  • obswl (Optional[Quantity]) – Observing wavelength.

  • pointing_model (Optional[str]) –

  • obsfreq (Optional[Quantity]) –

Variables:

  • location (EarthLocation) – Location of observatory.

  • pressure (Quantity) – Atmospheric pressure, to compute diffraction correction.

  • temperature (Quantity) – Temperature, to compute diffraction correction.

  • relative_humidity (Quantity or float) – Relative humidity, to compute diffraction correction.

  • obswl (Quantity) – Observing wavelength, to compute diffraction correction.

  • obsfreq (Quantity) – Observing frequency of EM-wave, to compute diffraction correction.

Examples

>>> location = EarthLocation("138.472153deg", "35.940874deg", "1386m")
>>> path = "path/to/pointing_param.toml"
>>> pressure = 850 * u.hPa
>>> temperature = 290 * u.K
>>> humid = 0.5
>>> obsfreq = 230 * u.GHz
>>> finder = neclib.coordinates.PathFinder(
...     location, path, pressure=pressure, temperature=temperature,
...     relative_humidity=humid, obsfreq=obsfreq)

functional(lon, lat, frame, *, unit=None, n_cmd, mode, time=None)[source]#

Calculate the path of antenna movement from any function.

Parameters:

  • lon (Callable[[float], T]) – Function that returns longitude. The function should accept single argument; auxiliary variable of value between 0 and 1.

  • lat (Callable[[float], T]) – Function that returns latitude.

  • frame (Union[str, BaseCoordinateFrame]) – Frame in which the coordinates are given.

  • unit (Optional[Union[str, UnitBase]]) – Angular unit in which longitude and latitude are given. If they are given as Quantity, this parameter will be ignored.

  • n_cmd (Union[int, float]) – Number of commands. The path will be calculated supplying arithmetic sequence of auxiliary variable, which first term is 0, last is 1, number of members is n_cmd.

  • time (Optional[Timer]) – Start time manager. If None, current time will be used. No consideration for antenna_command_offset_sec is required.

  • mode (ControlStatus) –

Returns:

Tuple of conversion result, azimuth, elevation, time and type information of the control section.

Return type:

Az, El, Time, ControlStatus

Examples

>>> def lon(x):
...     return 30 + x * 2
>>> def lat(x):
...     return 50
>>> path = finder.functional(lon, lat, frame="altaz", unit="deg", n_cmd=1000)
>>> next(path)
(<Quantity [30., 30.1, 30.2, ...] deg>, <Quantity [50., 50., 50., ...] deg>,
[1610612736.0, 1610612736.1, 1610612736.2, ...])
