class CoordCalculator(location, pointing_param_path, *, pressure=None, temperature=None, relative_humidity=None, obswl=None, obsfreq=None)[source]¶

Bases: object

Calculate horizontal coordinate.

Parameters

  • location (EarthLocation) – Location of observatory.

  • pointing_param_path (Union[PathLike, str]) – Path to pointing parameter file.

  • pressure (Quantity) – Atmospheric pressure at the observation environment.

  • temperature (Quantity) – Temperature at the observation environment.

  • relative_humidity (Union[int, float, ndarray, Quantity]) – Relative humidity at the observation environment.

  • obswl (Quantity) – Observing wavelength.

  • obsfreq (Quantity) –

Variables

  • location (EarthLocation) – Location of observatory.

  • pressure (Quantity) – Atmospheric pressure, to compute diffraction correction.

  • temperature (Quantity) – Temperature, to compute diffraction correction.

  • relative_humidity (Quantity or float) – Relative humidity, to compute diffraction correction.

  • obswl (Quantity) – Observing wavelength, to compute diffraction correction.

  • obsfreq (Quantity) – Observing frequency of EM-wave, to compute diffraction correction.

Return type

None

Examples

>>> location = EarthLocation("138.472153deg", "35.940874deg", "1386m")
>>> path = "path/to/pointing_param.toml"
>>> pressure = 850 << u.hPa
>>> temperature = 300 << u.K
>>> humid = 0.30
>>> obswl = 230.5 << u.GHz
>>> calculator = neclib.coordinates.CoordCalculator(
...     location, path, pressure=pressure, temperature=temperature,
...     relative_humidity=humid, obswl=obswl)

get_altaz(lon, lat, frame, *, unit=None, obstime)[source]¶

Get horizontal coordinate from longitude and latitude in arbitrary frame.

Parameters

  • lon (T) – Longitude of target.

  • lat (T) – Latitude of target.

  • frame (Union[str, BaseCoordinateFrame]) – Coordinate frame, in which lon and lat are given.

  • unit (Optional[Union[str, Unit]]) – Angular unit in which lon and lat are given. If they are given as Quantity, this parameter will be ignored.

  • obstime (Union[int, float, ndarray, Time]) – Time the observation is done.

Return type

SkyCoord

Examples

>>> calculator.get_altaz(30 << u.deg, 45 << u.deg, "fk5", obstime=time.time())
<SkyCoord (az, alt) in deg (344.21675916, -6.43235393)>
