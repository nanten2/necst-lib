neclib.parameters.pointing_error¶

Pointing error correction.

The telescope pointing is critically important in performing accurate and uniform observation. This module employs following pointing model:

\[\begin{split}\Delta x =& \chi_{Az} \sin ( \omega_{Az} - Az ) \sin ( El ) \\ &+ \epsilon \sin ( El ) \\ &+ \chi_{2, Az} \sin ( 2 ( \omega_{2, Az} - Az ) ) \sin ( El ) \\ &+ \mathrm{d}Az \cos ( El ) \\ &+ \mathrm{d}e \\ &+ \mathrm{cor}_v \cos ( El + \mathrm{cor}_p ) \\ &+ \mathrm{d}e_\mathrm{radio} \\ \Delta Az =& \Delta x / \cos ( El ) \\ \Delta y =& - \chi_{El} \cos ( \omega_{El} - Az ) \\ &- \chi_{2, El} \cos ( 2 ( \omega_{2, El} - Az ) ) \\ &+ g_1 \cos ( El ) + g_2 \sin ( El ) \\ &+ \mathrm{d}el \\ &+ g_{ 1,\mathrm{radio} } \cos ( El ) + g_{ 2,\mathrm{radio} } \sin ( El ) \\ &- \mathrm{cor}_v \sin ( El + \mathrm{cor}_p ) \\ &+ \mathrm{d}el_\mathrm{radio} \\ \Delta El =& \Delta y\end{split}\]
