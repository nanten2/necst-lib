class CPZ7415V[source]#

Bases: PulseController

Pulse controller, which can handle up to 4 motors (axes).

  • rsw_id

    Board identifier, set by rotary switch with label “RSW1” mounted on the side of the board. The board is shipped with default RSW1 setting of 0. This ID would be non-zero, when multiple PCI board of same model are mounted on a single FA (Factory Automation) controller. Accepted values are [0, 1, ..., 16] if given as int, ["0", "1", ..., "9", "A", ..., "F"] if str.

  • other parameters

    See <http://www.interface.co.jp/download/tutorial/tut0053_14.pdf>.
